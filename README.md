# xukaigao.github.io

个人主页（GitHub Pages · 纯 HTML/CSS）。

## 发布步骤

1. 在 GitHub 新建 **公开** 仓库，名称必须为：**`xukaigao.github.io`**。
2. 将本仓库文件推到 `main` 分支根目录（包含 `index.html`）。
3. GitHub 仓库 → **Settings → Pages**：**Branch** 选 `main`，**Folder** 选 `/ (root)`，保存。
4. 等待约 1～3 分钟后访问：<https://xukaigao.github.io>

## 本地预览（可选）

直接用浏览器打开 `index.html`，或使用任意静态服务器。
