你好，我是 xukaigao

这是用静态 HTML/CSS 搭建的个人主页示例。你可以在本地修改 index.html 与 style.css，推送到 GitHub 后自动更新站点。

链接

  • GitHub 主页
  • 站点地址：https://xukaigao.github.io

接下来你可以

  1. 在 GitHub 新建仓库 xukaigao.github.io（公开）。
  2. 把本目录文件推到 main 分支根目录。
  3. 仓库 Settings → Pages：Branch 选 main，文件夹 / (root)。
  4. 等待几分钟，访问本站域名。
